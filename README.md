# ZM79-HTML-Video-Project-Kit
This is a simple html video project kit that is perfect for archiving your old YouTube/Dailymotion/Vimeo/TikTok/Instagram accounts, for special video projects, personal vlogs, school html assignments for teachers, digital tv shows for those that don't want to bother with big tech censorship or their distractions, whatever you need this kit for. The kit is also completely free to use and modify and the background css and text css can have their colors modified to make the project suitable for what they need it for.

By using this kit, you agree not to host any copyrighted content with it or anything that would otherwise violate your local laws. Furthermore, use of this kit is best suited for self-hosting (e.g., a web domain via cloudflare), as while you can certainly make your project with this on github, they have a limit of 1gb for each project. Using this outside github is recommended for that reason.
